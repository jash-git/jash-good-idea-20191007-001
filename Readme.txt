一位年輕人為了逃避兵役，竟然娶了比自己大57歲的阿婆當新娘。

雖然這名年輕人的作為備受批評，但法律似乎是站在他那邊。

媒體《T1》報導，來自烏克蘭文尼察市的Alexander Kondratyuk在2017年底收到了兵單，得去當地徵兵處報到。

在烏克蘭，所有年齡介於18歲到26歲間的年輕男子，都得服一年兵役。

除非該男子必須照顧身體殘疾的太太，才能免除兵役義務。

Alexander不知用什麼方式說服了他81歲的表姊跟他結婚。

才24歲的Alexander已經與表姊結婚2年了。

再過2年，Alexander就超過會被徵兵的年紀了，到時候他有可能會離婚。

這起案件浮上檯面，是因為2018年文尼察市軍事單位向檢察官提告，要取消這對夫婦的婚姻。

軍方認為Alexander是偽造文件以逃避兵役。然而經過查證，這對夫婦確實已合法結婚。

市政單位也確認他們的婚姻是合法的，因此軍方不能徵召Alexander入伍。

今年9月，這起案件又在烏克蘭媒體上被爆出，因為媒體向軍方詢問此案的資訊。

文尼察負責徵兵的軍官表示，Alexander的文件確實是合法的。

Alexander本人也一再否認他是為了逃避兵役，才娶了大他57歲的表姊為妻。

他聲稱自己對Zinaida有很深的感情，然而Zinaida的鄰居卻表示Alexander根本是在說謊。

鄰居表示Zinaida獨自居住，而她的孫子和外甥比Alexander還常來探望她。

據《Metro》報導，這起案件仍在調查中，然而軍方表示他們並沒有興趣深究此案。

由於Alexander已出具正式文書，證明他已婚且住在太太家裡，軍方已無法多做什麼。

然而法律專家表示，他們其實可以傳喚周遭鄰居作為證人，證明Alexander是否真住在該住屋裡。

若能證明他們的婚姻並不真實，且居住事實與宣稱不符，那麼Alexander仍可能會被徵召入伍。
